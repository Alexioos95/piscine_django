Ex00: Markdown Cheatsheet

Headings

			
				# H1
				## H2
				### H3
				#### H4
				##### H5
				###### H6

Emphasis

			
				*italic* or _italic_
				**bold** or __bold__
				***bold italic*** or ___bold italic___
				~~strikethrough~~

Lists

Unordered

			
				- Item A
				- Item B
					- Subitem B.1
				(May also work with * instead of -)

Ordered

			
				1. First
				2. Second
					1. Sub-item

Links

			
				[Text](https://www.example.com)

Code

Inline

			
				print()

Block

			
				def function():
						instruction

Blockquotes

			
				> This is a blockquote.
				>> It can span multiple lines.

Horizontal Rule

			
				---
				or
				***

Tables

			
				| Header 1| Header 2|
				|---------|---------|
				|   Value1|   Value1|
				|   Value2|   Value2|
